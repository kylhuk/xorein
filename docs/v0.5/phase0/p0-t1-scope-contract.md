# Phase 0 · P0-T1 Scope Contract

## Goal
Anchor v0.5 execution to the eight roadmap bullets listed in Section 2 of `TODO_v05.md` and ensure each has an auditable trace to a phase task and validation artifact.

## Contract
- Each scope bullet from `S5-01` through `S5-08` is mapped to a Phase 1–6 task and a `VA-*` artifact before V5-G0 exits; the trace table in Section 8 of `TODO_v05.md` provides the canonical mapping.
- Scope documentation must cite the exclusion rules from Section 3 and the carry-back dependency rule in Section 4.5 so that no v0.6+ capability or unlocked prerequisite is implicitly folded into v0.5.
- Compliance evidence consists of this contract, the compatibility/governance checklist in `docs/v0.5/phase0/p0-t2-compatibility-governance-checklist.md`, and the release-gate handoff pack in `docs/v0.5/phase6/p6-t3-release-gate-handoff.md`.
